## use

Application can be viewed as GitHub Pages under: https://steffstefferson.github.io/azores_trails_onamap

Features:

Since there was no handy map on the offical site https://trails.visitazores.com/ to explore the hikes by area/region, here it is.

## develop

No build step required, page can be viewed with a python server by exec the following command in the src directory: python -m http.server
